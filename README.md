## Descrição do projeto
Este projeto é para a disciplina de Desenvolvimento Web Front-end do curso de Análise e Desenvolvimento de Sistemas que cursamos no Instituto Federal de Educação, Ciência e Tecnologia de São Paulo - Câmpus Bragança Paulista.
O SecureKey é um sistema de gerenciamento de senhas inspirado no [BitWarden](https://bitwarden.com/). Este sistema tem o objetivo de facilitar o gerencimento de senhas que o usuário possui.

## Status do projeto
🚧 Em desenvolvimento 🚧

## 🔨 Funcionalidades
- Salvar senhas.
- Associar as senhas a sites.
- Pesquisar senhas.
- Gerar senhas.

## Como instalar e executar o projeto.

Abra o terminal e clone o repositório.
```bash
  git clone https://github.com/GustavoBertuzzi/Securekey.git
```

Mude de pasta.
```bash
  cd securekey
```

Instale o node modules.
```bash
  npm install
```

Execute localmente
```bash
  npm install
```
